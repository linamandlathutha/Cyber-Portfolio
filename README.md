# Linamandla // Cybersecurity Analyst Portfolio

A high-contrast, monochrome, tactical telemetry interface mapping defensive operations, threat vectors, and custom automation infrastructure. Designed with a clean IDE-inspired layout using premium, structured typography.

## 🛠️ Tech Stack & Architecture
* **Frontend Matrix:** Semantic HTML5, Vanilla CSS3 (Advanced Grid, Flexbox alignment, Custom Properties)
* **Fluid Geometry:** Responsive scaling using the CSS `clamp()` engine to guarantee absolute cross-device stability.
* **Telemetry & Automation:** Vanilla JavaScript DOM manipulation, custom multi-stage Typewriter timing loops, and Intersection Observer API for fluid interface rendering.
* **Data Processing:** Serverless secure uplink integration powered by Web3Forms for direct-to-operator ingestion routing.
* **Typography Elements:** `Basote` (Display/Headers) | `Poppins` (Body Readout).

## 📊 Core Interface Breakdown
* **index.html:** Main tactical command center featuring an adaptive grid layout, animated headline statement, asset boundaries for high-resolution telemetry mapping ($1536 \times 1024$ and $2048 \times 1948$), and custom HUD profile framing.
* **experience.html:** Chronological multi-stage vertical deployment history tracking enterprise control evaluation metrics and active security tool tracking arrays.
* **projects.html:** Threat mitigation registry logging incident response protocols for High and Critical security alerts alongside an interactive classification triage console.
* **contact.html:** Secure communications link node equipped with interactive bracket CSS animation indicators and secure form routing.
